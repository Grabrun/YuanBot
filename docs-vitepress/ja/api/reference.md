# APIリファレンス

> 🚧 **このページは準備中です。**

YuanBot はRESTful APIとWebSocketエンドポイントを提供します。

**ベースURL**: `http://localhost:8000`

## エンドポイント一覧

| カテゴリ | プレフィックス | 説明 |
|----------|--------|------|
| 認証 | `/api/auth` | ログイン、ログアウト |
| チャット | `/api/chat` | メッセージ送信 |
| ペルソナ | `/api/persona` | ペルソナ管理 |
| Provider | `/api/providers` | AIプロバイダー管理 |
| メモリ | `/api/memory` | メモリ照会 |
| TTS | `/api/tts` | 音声合成 |
| ヘルス | `/healthz` | ヘルスチェック |

詳細については、[中国語ドキュメント](/api/reference)をご参照ください。
