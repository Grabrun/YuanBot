## 前提条件

- Python 3.12+
- pip または uv
- AIプロバイダーのAPIキー

## インストール

```bash
pip install yuanbot

# すべての拡張機能を含める場合
pip install "yuanbot[all]"
```

## 起動

```bash
yuanbot start
```

詳細については、[中国語ドキュメント](/guide/getting-started)をご参照ください。
