## 前提条件

- Python 3.12+
- pip または uv
- AIプロバイダーのAPIキー

## インストール

```bash
git clone https://github.com/Grabrun/YuanBot.git
cd YuanBot
python3 -m venv .venv     # 没反应? sudo apt install python3-venv python3-full -y
source .venv/bin/activate
pip install -e ".[dev]"
yuanbot version
```

## 起動

```bash
yuanbot start
```

詳細については、[中国語ドキュメント](/guide/getting-started)をご参照ください。
