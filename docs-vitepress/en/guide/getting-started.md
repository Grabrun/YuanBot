# Quick Start

> 🚧 **This page is a work in progress.** English documentation is being translated.

## Prerequisites

- Python 3.12+
- pip or uv
- AI Provider API Key

## Installation

```bash
pip install yuanbot

# or with all extras
pip install "yuanbot[all]"
```

## Quick Start

```bash
yuanbot start
```

For detailed information, please refer to the [Chinese documentation](/guide/getting-started).
