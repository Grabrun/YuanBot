# Installation

> 🚧 **This page is a work in progress.** English documentation is being translated.

## Requirements

- Python 3.12+
- pip / uv
- Docker (optional)

## via pip

```bash
pip install "yuanbot[all]"
```

## via Docker

```bash
git clone https://github.com/Grabrun/YuanBot.git
cd YuanBot
cp .env.example .env
docker-compose up -d
```
